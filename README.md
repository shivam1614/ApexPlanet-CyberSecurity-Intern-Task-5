# ApexPlanet-CyberSecurity-Intern-Task-5
Task 5 Submission


Security & Ethical Hacking Internship Program at ApexPlanet Software Pvt. Ltd.

The objective of this project was to conduct a comprehensive penetration test on DVWA (Damn Vulnerable Web Application) in a controlled laboratory environment. The assessment focused on identifying, analyzing, and documenting common web application vulnerabilities while recommending appropriate mitigation strategies.

Objectives
Perform a security assessment of DVWA.
Identify vulnerabilities based on OWASP Top 10.
Analyze the impact of discovered vulnerabilities.
Demonstrate exploitation in a controlled environment.
Recommend remediation techniques.
Prepare a professional penetration testing report.
